# Recruti-tn

## Technologies Used
- Angular: Frontend  
- Spring Boot: Backend  
- MySQL: Database  
- Docker & Docker Compose: Containerization  
## How to run the project: 
```bash
docker-compose -f docker-compose.dev.yml up

to build a particular service (eg. frontend) :
```bash
docker-compose -f docker-compose.dev.yml up -build frontend

Please make sure you are running this command from the correct directory (where the file is located).
